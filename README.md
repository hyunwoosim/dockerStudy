# DockerStudy
1. npm init -y
2. package.json 자동 생선된다
   - 라이브러리에 무엇을 설치했는지 기록하는 용도의 파일(라이브러리 기록용 파일)
3. npm install express
  - 서버를 쉽게 만들어주는 라이브러리
4. node server.js
  - 서버 실행 
5. DockerFile 생성
  - dockerImage를 만들기 위한 용도



## Tip
1. nodemon
  - 코드를 실시간 반영해주는 라이브러리 
  - npm install -g nodemon 설치
  - nodemon server.js 실행
2. pm2
  - 서버가 에러 때문에 죽었을때 알아서 실행되도록 해주는 라이브러리
